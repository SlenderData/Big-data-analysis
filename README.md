# BigDataAnaylsis

本项目为我个人于 2021 - 2022 学年第 2 学期在学习课程 **Python大数据分析与应用** 时所编写的代码及参考的资料。

## 运行环境

- **操作系统**
  - macOS Monterey 12.3.1
- **Python环境**
  - Conda (Python 3.10)
- **使用到的软件**
  - [PyCharm](https://www.jetbrains.com/pycharm/)
- **额外的软件包**
  - flask
  - lxml
  - bs4
    - beautifulsoup4
  - scrapy
  - itemadapter
